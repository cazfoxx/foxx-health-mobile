# Foxx Health Frontend


## Technical Stack

- **Framework**: Flutter
- **Authentication**: Firebase Auth
- **State Management**: Flutter Bloc
  
 ## Installation
- git clone https://github.com/foxxhealth/frontend.git
- flutter pub get
- flutter run
